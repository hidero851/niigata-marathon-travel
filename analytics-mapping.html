SLIDE 07
GA4での確認方法
KPI別のレポート操作ガイド
KPI別・見るべきレポート
見たいKPI GA4の場所 確認するイベント
どの大会が人気か レポート → エンゲージメント → イベント view_event_detail（event_idで絞る）
エントリー誘導率 レポート → エンゲージメント → イベント click_entry_link / click_entry_final_cta
楽天リンクのCTR レポート → エンゲージメント → イベント click_rakuten_*（複数）
どの検索条件が多いか レポート → エンゲージメント → イベント search（パラメータ: tags）
人気タグ・フィルター レポート → エンゲージメント → イベント filter（パラメータ: tags）
特産品の購買意欲 レポート → エンゲージメント → イベント click_product_external
流入元（SNS/検索等） レポート → 集客 → トラフィック獲得 セッションのデフォルト チャネル グループ
リアルタイム動作確認 レポート → リアルタイム 全イベント（操作と同時に反映）
パラメータの確認方法
GA4でイベント名をクリック
→ 「パラメータ」タブで
　event_id ごとの内訳が確認できる。

ただし カスタムディメンション を設定しないとレポートに表示されない場合がある。

設定手順：
管理 → カスタム定義
→ 「カスタムディメンション」を作成
→ スコープ: イベント
→ イベントパラメータ: event_id
ローカルログの確認：
/datasource ページ（管理画面）でlocalStorageに蓄積したログを一覧表示できる。
